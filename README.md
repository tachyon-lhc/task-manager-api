# Task Manager API

A RESTful API for managing tasks, built with Flask and SQLite. This project was done with the goal of learning about API development, testing, CRUD operations, and clean architecture.

![Tests](https://img.shields.io/badge/tests-15%20passed-brightgreen)
![Python](https://img.shields.io/badge/python-3.8+-blue)
![Flask](https://img.shields.io/badge/flask-3.0.0-lightgrey)

## Features

- **Full CRUD Operations** - Create, read, update, and delete tasks
- **RESTful API Design**
- **SQLite Database**
- **Task Priorities** - Organize tasks by priority (low, medium, high)
- **Timestamps** - Automatic creation and update timestamps
- **Input Validation** - Basic validation for required fields
- **Comprehensive Testing** - 15+ unit and integration tests with pytest
- **Test Coverage** - Separate tests for models and API endpoints

## Tech Stack

- **Language:** Python 3.x
- **Framework:** Flask
- **Database:** SQLite
- **Testing:** Pytest
